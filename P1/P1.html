以SQL获取需要数据：¶

SELECT c.year,c.city,c.country,c.avg_temp as avg_cd,g.avg_temp as avg_temp_gloab
FROM city_data c 
JOIN global_data g on c.year=g.year
WHERE c.city='Chengdu'
--AND c.avg_temp is not null 因为时间是continous data
